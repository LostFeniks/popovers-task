# Popover Widget

[![Build and Deploy](https://github.com/LostFeniks/popovers-task/actions/workflows/build.yml/badge.svg)](https://github.com/LostFeniks/popovers-task/actions/workflows/build.yml)

## Описание

Виджет Popover на чистом JavaScript без использования jQuery. Реализован по аналогии с Bootstrap Popover.

## Особенности

- Чистый JavaScript без зависимостей
- Автоматическое позиционирование сверху от элемента
- Центрирование по горизонтали
- Анимация появления
- Закрытие при клике вне popover
- Адаптивная верстка
- Webpack сборка
- Автоматический деплой на GitHub Pages
